# Portfolio

Website portfolio pribadi dan CV printable.

## Cara Menjalankan

1. Install Node.js.
2. Install dependencies:
   ```bash
   npm ci
   ```
3. Jalankan mode development:
   ```bash
   npm run dev
   ```
4. Buka browser:
   ```text
   http://localhost:3000
   ```

## Cek Sebelum Upload

Jalankan perintah berikut:

```bash
npm run lint
npm run build
```

## Fitur

- Portfolio responsive
- Dark mode dan light mode
- Form contact via email
- Modal CV
- Print atau save CV ke PDF

## Teknologi

- React
- TypeScript
- Vite
- Tailwind CSS
