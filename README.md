# Ade Iqbal Junianto Portfolio

Website portfolio pribadi dan CV printable untuk Ade Iqbal Junianto.

## Cara Menjalankan

1. Install Node.js.
2. Install dependencies:
   ```bash
   npm ci
   ```
3. Jalankan mode development:
   ```bash
   npm run dev
   ```
4. Buka browser:
   ```text
   http://localhost:3000
   ```

## Cek Sebelum Upload

Jalankan perintah berikut:

```bash
npm run lint
npm run build
```

## Fitur

- Portfolio responsive
- Konten bahasa Indonesia dan Inggris
- Dark mode dan light mode
- Tombol kontak WhatsApp dan email
- Bagian proyek pilihan
- Metadata SEO dan social preview
- Modal CV
- Print atau save CV ke PDF

## Teknologi

- React
- TypeScript
- Vite
- Tailwind CSS
